Mini Sistema de Estoque com CRUD em Python

Projeto desenvolvido para praticar lógica de programação, funções, listas, dicionários e operações básicas de CRUD.

Funcionalidades

- Cadastrar produto
- Listar produtos
- Buscar produto por código
- Atualizar produto
- Remover produto
- Exibir relatório de estoque baixo

Objetivo do projeto

Este projeto foi criado como exercício prático para reforçar conceitos básicos de programação e simular um pequeno sistema de controle de estoque via terminal.
